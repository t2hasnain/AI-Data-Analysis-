Title: AI-Powered Data Analysis Assistant
Subtitle: Track A – Explorer Challenge Submission

1. Project Overview
-------------------
• What it is: A fully automated data analysis platform built with Python and Streamlit.
• Purpose: To allow users without coding experience to upload datasets and instantly receive insights, visualizations, and an AI-driven Q&A experience.
• Target Users: Business analysts, students, and data enthusiasts.

2. Core Features (Required Elements)
------------------------------------
• Dataset Loader: Supports CSV, JSON, Excel (.xlsx), and Parquet. Instantly reads data and displays size, types, and missing values.
• Statistical Engine: Calculates count, mean, min, max, and frequency using Pandas.
• Automated Visualizations: Dynamically generates Bar Charts, Pie Charts, Histograms, and Correlation Heatmaps using Plotly.
• AI Natural Language Q&A: Uses an LLM (Azure OpenAI, Gemini, or Groq via LangChain) to answer fixed or custom questions accurately based on dataset context.
• AI Explanation: Generates easy-to-understand narrative summaries of the charts and statistical trends.

3. Bonus Features & Enhancements
--------------------------------
• UI/UX Excellence: Custom Light/Dark mode toggles with a premium design system.
• Multi-Format Upload: Beyond CSV, handles enterprise formats like Excel, JSON, and Parquet natively using Pandas.
• Chatbot Widget: Floating "DataBot" for interactive, persistent conversations about the data.
• PDF Export: One-click "Export Analysis to PDF" to generate a professional offline report (utilizes custom formatting to guarantee PDF generation even with emojis).
• Caching & Optimization: Uses Streamlit session_state to prevent redundant API calls, saving costs and improving speed.

4. Technology Stack
-------------------
• Frontend: Streamlit (Python UI Framework) + Custom CSS
• Backend: Python 3.11, Pandas, NumPy
• AI/LLM: Azure OpenAI, Gemini, Groq, LangChain
• Visualizations: Plotly Express
• Exporting: FPDF2

5. Accuracy & Compliance
------------------------
• The AI assistant is highly accurate because it is provided with the exact statistical summaries, column schemas, and sample data in its prompt context. The AI does not hallucinate numbers because the mathematical aggregations (Min, Max, Mean) are computed strictly by Pandas before the AI sees them.
• All rules from the challenge (required.txt) have been strictly followed: pure Python logic, LLM integration, one meaningful chart (plus extra options), and clear explanations.

6. Conclusion
-------------
• This project bridges the gap between raw data and actionable intelligence, satisfying all Track A Explorer requirements while pushing the boundaries with advanced UI, multiple formats, Docker support, and multi-modal export features.
